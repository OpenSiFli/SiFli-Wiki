# LCD frame-rate calculator

* Use the LCD frame-rate calculator to calculate the display refresh rate.<br>

<a href="lcd_frequency.html" target="_blank">Click here to open the LCD frame-rate calculator</a> <br>

* This calculator can help you:
    - Calculate the refresh rate (FPS) of the LCD display.
    -  Calculate the time required for each refresh (ms).<br>

* The following parameters must be entered during use:
    - Resolution width (pixels)
    - Resolution height (pixels)
    - Color bit depth (RGB565 is 16-bit, RGB888 is 24-bit)
    - Interface frequency (Hz)
    - DDR mode (enter 1 for single data rate, 2 for double data rate)
