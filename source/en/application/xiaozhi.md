# AI Xiaozhi

AI Xiaozhi is an AI voice assistant solution for SiFli chips. It provides features such as speech recognition and speech synthesis, and is suitable for application scenarios such as smart homes and smart speakers.

The currently implemented features are:
- Voice wake-up KWS
- Continuous dialog
- AEC and VAD
- Dynamic expressions
- Vector fonts
- Local MCP
