# Software Debugging-Related Issues

```{important}
SiFli-series MCUs can use the following debugging methods:
* Log Printing Debugging Method
* Jlink, Ozone, UartService, and SystemView Online Debugging Methods
* Dump Memory Offline Debugging Method
* Crash Context Saving Method
* UI Simulator Debugging Method
```
