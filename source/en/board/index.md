# Development board

## Description

SiFli Technologies Co., Ltd. (SiFli) provides multiple development boards for different application scenarios and requirements. The materials on this site are mainly intended for development boards sold publicly. For development boards used for internal testing or not sold publicly, contact the sales representative from whom you obtained the development board to obtain the related materials.

**Key Features:**
- Integrated download and flashing function

- Standard FPC interface for display expansion

A compact development board designed based on the SF32LB52 chip, suitable for use as a core board or for embedded application development.

**Key Features:**
- Compact size of only 21 mm x 51 mm
- ENIG castellated-hole design for easy soldering
- FPC interface for display expansion
- All core devices are integrated, enabling use as a core board

A core development board designed around the SF32LB52 chip, with rich GPIOs and an FPC display interface for prototyping and power testing.

**Key Features:**
- Compact 30.88 mm x 68 mm core-board form factor
- Castellated through-hole header design for solder-free debugging
- Supports 1.8 V/3.3 V jumper power configuration
- Supports single-rail and whole-board power consumption tests
